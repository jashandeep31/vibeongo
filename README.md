# Vibeongo

Vibeongo is a cloud workspace for building software with AI coding assistants.
Connect a GitHub repository, start an isolated development environment, preview
changes live, and move the work back into your GitHub workflow.

This repository contains the core runtime and platform code for Vibeongo.

## What It Does

- Connects with GitHub repositories
- Starts ready-to-code cloud workspaces
- Supports AI-assisted development workflows
- Provides live previews for running projects
- Helps move changes back into commits and pull requests
